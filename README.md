## Tokens de personalización HubSpot

| Token | Qué inserta |
|---|---|
| `[Nombre]` | Primer nombre del contacto |
| `[Empresa]` | Nombre de la empresa |
| `[Industria]` | Sector (ej. hotelería, logística) |
| `[Fecha]` | Fecha de reunión agendada |
| `[Hora]` | Hora de la reunión |
| `[Producto/Servicio]` | Lo que se discutió en reunión |
